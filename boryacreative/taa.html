FAQ

Турнир: Арена Адаптации — инструмент для стримера Инструкцию ИИшка писала если чо AHAH WW

Что это?

Веб-инструмент для проведения ивента "Турнир: Арена Адаптации".

Режим подготовки

Заполните имена участников и добавьте игры в каждую из трёх категорий. Названия категорий стримеров генерируются автоматически из имён. Нейтральные игры можно переименовать вручную.

Готовый список можно экспортировать в JSON-файл (кнопка "Экспорт JSON"). Этот файл потом загружается в боевом режиме. Ранее сохранённый список можно импортировать через "Импорт JSON" для редактирования.

Боевой режим

Загрузите JSON-файл, созданный в режиме подготовки, через drag'n'drop или кнопку выбора файла.

После загрузки нажимайте в любом месте экрана — игры будут открываться по одной с анимацией печатающегося текста. Порядок раскрытия: сначала нейтральные, потом игры стримеров.

Когда все игры раскрыты, откроется обзорный экран с тремя колонками. У каждой игры есть две кнопки по краям: красная — победа первого стримера, синяя — победа второго. Клик по кнопке отмечает победителя, повторный клик снимает отметку.

Снизу отображается счёт с количеством выигранных игр у каждого участника.

Настройки оформления

Во вкладке "Настройки" можно изменить внешний вид под своё оформление. Доступны три блока:

  • Основные — шрифт (семейство и размеры), цвет фона, фоновое изображение, размер заголовков и игр в списках, текст боевого режима.
  • Дополнительные — цвет текста, акцентный цвет, затемнение фона, скругление углов.

Изменения сохраняются в браузере и восстанавливаются при повторном открытии страницы.

Как сохранить конфигурацию с играми?

В режиме подготовки нажмите "Экспорт JSON" — браузер скачает файл taa_config.json. Храните его рядом. Для использования в новом стриме просто загрузите этот файл в боевом режиме.

Мини-игры и синхронизация

После начала боевого режима в левом нижнем углу появляется панель с кнопками:

  • Мини-игры — монетка, кубик, камень-ножницы-бумага. Результаты синхронизируются между стримерами.
  • Колесо игр — выбор игр галочками, прокрутка с анимацией, сыгранные игры блокируются.
  • Таймер — можно настроить время таймера, который будет отображаться в виджете, запуск/остановка синхронизируется.

Все действия пишутся в общий лог (мини-лог над кнопками, полный лог по клику).

OBS-виджеты

Для вывода счёта и таймера в OBS используйте Browser Source с URL вида:

  • widget-score.html?room=КОД — счёт игр
  • widget-timer.html?room=КОД — таймер обратного отсчёта

КОД — это код комнаты, отображаемый на сайте после создания/подключения. Кнопки копирования этих URL находятся справа снизу в боевом режиме после входа в комнату.

Как пользоваться сайтом

Режим подготовки

  1. Введите имена стримеров и название нейтральной категории
  2. Добавьте игры в каждую категорию (нейтральные / стример №1 / стример №2)
  3. Нажмите «Экспорт JSON» — сохраните файл для боевого режима
  4. Ранее сохранённый JSON можно импортировать через «Импорт JSON»

Боевой режим

  1. Перетащите JSON-файл в зону загрузки или нажмите «Выбрать файл»
  2. Кликайте по экрану — игры открываются с анимацией по одной
  3. Порядок: нейтральные, потом игры стримеров по очереди
  4. После раскрытия всех — экран обзора с тремя колонками
  5. Кликайте по кнопкам слева/справа от игры чтобы отметить победителя
  6. Снизу отображается счёт, справа — блок «Порядок матчей»

Синхронизация с другим стримером

  1. Загрузите JSON → нажмите «Создать комнату»
  2. Отправьте 4-символьный код оппоненту (КОД НЕ ПОКАЗЫВАТЬ НА СТРИМЕ)
  3. Оппонент вводит код и нажимает «Присоединиться»
  4. Нажмите «Начать» P.S. игры не раскрываются синхронно, каждый раскрывает их сам
  5. Если планируете пользоваться минииграми, ОБЯЗАТЕЛЬНО снизу слева в выпадающем списке отметьте кто вы есть
  6. Отмечайте победителей — счёт и список побед синхронизируются
  7. Кнопка «Синхронизировать» принудительно обновляет данные
  8. Кнопка «Заново» сбрасывает комнату и возвращает к загрузке

Мини-игры и прочие штуковины

На случай если нужно принять решение, а оно не принимается xdd

Панель с мини-играми появляется слева внизу после начала боевого режима.

  • Монетка — бросьте монетку, результат запишется в лог
  • Кубик — выберите количество граней (d6/d12/d20/d228/d1137), бросьте
  • КНБ — камень-ножницы-бумага: выберите фигуру, подождите выбора оппонента, после чего определится победитель
  • Колесо игр — отметьте игры галочками и крутите. Сыгранные игры выбрать для колеса нельзя
  • Таймер — введите время (минуты:секунды), нажмите «Запустить». Таймер синхронизируется и появляется в виджете. Кнопка «Стоп» сбрасывает

Все действия пишутся в лог — мини-лог над кнопками (последние 5 событий), полный лог открывается по клику на мини-лог.

OBS-виджеты

Для вывода счёта и таймера в OBS используйте Browser Source:

  • widget-score.html?room=КОД — счёт игр (имена + очки)
  • widget-timer.html?room=КОД — таймер обратного отсчёта (по дефолту скрыт, для настройки можно запустить таймер, чтобы он отображался в виджете)

URL можно скопировать кнопками над «Синхронизировать» в боевом режиме.

Emote Rain

В настройках укажите свой Twitch-канал и нажмите «Запустить дождь». С каждым отправленным в чат эмоутом на экран падают 7TV-эмоуты. Эмоуты подгружаются и из совместного чата (Shared Chat).
