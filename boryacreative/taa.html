FAQ

Турнир: Арена Адаптации — инструмент для стримера Инструкцию ИИшка писала если чо AHAH WW

Что это?

Веб-инструмент для проведения ивента "Турнир: Арена Адаптации".

Режим подготовки

Заполните имена участников и добавьте игры в каждую из трёх категорий. Названия категорий стримеров генерируются автоматически из имён. Нейтральные игры можно переименовать вручную.

Готовый список можно экспортировать в JSON-файл (кнопка "Экспорт JSON"). Этот файл потом загружается в боевом режиме. Ранее сохранённый список можно импортировать через "Импорт JSON" для редактирования.

Боевой режим

Загрузите JSON-файл, созданный в режиме подготовки, через drag'n'drop или кнопку выбора файла.

После загрузки нажимайте в любом месте экрана — игры будут открываться по одной с анимацией печатающегося текста. Порядок раскрытия: сначала нейтральные, потом игры стримеров.

Когда все игры раскрыты, откроется обзорный экран с тремя колонками. У каждой игры есть две кнопки по краям: красная — победа первого стримера, синяя — победа второго. Клик по кнопке отмечает победителя, повторный клик снимает отметку.

Снизу отображается счёт с количеством выигранных игр у каждого участника.

Настройки оформления

Во вкладке "Настройки" можно изменить внешний вид под своё оформление. Доступны три блока:

  • Основные — шрифт (семейство и размеры), цвет фона, фоновое изображение, размер заголовков и игр в списках, текст боевого режима.
  • Дополнительные — цвет текста, акцентный цвет, затемнение фона, скругление углов.

Изменения сохраняются в браузере и восстанавливаются при повторном открытии страницы.

Как сохранить конфигурацию с играми?

В режиме подготовки нажмите "Экспорт JSON" — браузер скачает файл taa_config.json. Храните его рядом. Для использования в новом стриме просто загрузите этот файл в боевом режиме.
